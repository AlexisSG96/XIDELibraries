System Module

Current System Clock

Summary: The microntroller clock frequency (Fosc).
Description: This is the clock frequency that the microcontroller core operates with. All timings are determined by this value. You can change its value using a combination of the settings below.
Additional information: This is not the clock frequency driven to peripherals, which is Fosc/4.
Availability: Always available. Not directly writable.

Oscillator Select

Summary: Defines via the Configuration Registers the oscillator used to generate the System Clock.
Description: A variety of clock sources is available for each device. The clock source list may vary from one device to another, but you will always have the basic choice between using internal or external clock sources. Selecting an internal clock disables the External Clock input box, while selecting an external clock disables the Internal Clock drop-down. Your selection is mapped into the FOSC setting found in the device Configuration Registers.
Additional Information: Each new selection triggers a Current System Clock update. All time-sensitive peripherals (i.e. EUSART, Timers, PWM) are notified by this change and, if needed, some timings are recalculated.
Availability: Always available.

System Clock Select

Summary: Override control for defining the system clock.
Description: Depending on the device, you have the choice to define your System Clock source from:

- the FOSC setting in the device Configuration Registers (selectable from the Oscillator Select comobobox or via the Registers View of MCC).

- the Internal Osccilator (INTOSC), via the IRCF setting in the Oscillator module .

- the Secondary Oscillator (SCS)

- the clock supplied to the Timer1 Oscillator circuitry
Additional Information: Each new selection triggers a Current System Clock update. All time-sensitive peripherals (i.e. EUSART, Timers, PWM) are notified by this change and, if needed, some timings are recalculated.
Availability: Always available. Options may vary.

Internal Clock

Summary: INTOSC clock frequency.
Description: Selects between several internal clock frequencies. Some devices offer multiple options for the same clock frequency which are generated using different INTOSC modes of operation. Most devices contain explicitly marked options that can be used together with an internal Phase Locked Loop (PLL) multiplier to reach higher clock frequencies.
Additional Information: Each new selection triggers a Current System Clock update. All time-sensitive peripherals (i.e. EUSART, Timers, PWM) are notified by this change and, if needed, some timings are recalculated.
Availability: The Internal Clock is disabled when the External Clock is enabled and viceversa.

External Clock

Summary: The clock frequency of the selected external clock source.
Description: If you select an external clock source via the Oscillator Select drop-down, the Internal Clock drop-down is disabled and the External Clock is enabled instead. You are required to input the external clock frequency you will be supplying on the oscillator pins. This is crucial for time-dependent peripherals and, though it may be left unchanged if you don't use time-dependent modules, you are strongly encouraged to fill it in to match your actual application. The input format is [Number][units], where units = Hz, kHz, MHz. Using a value greater than the device maximum frequency will result into a warning in the Notifications tab. Using an input string non-compliant with this format will yield the input box invalid and blinking red until the input becomes valid again or the focus is shifted.
Additional Information: Each new selection triggers a Current System Clock update. All time-sensitive peripherals (i.e. EUSART, Timers, PWM) are notified by this change and, if needed, some timings are recalculated.
Availability: The Internal Clock is disabled when the External Clock is enabled and viceversa.

PLL Enabled

Summary: Enables the PLL via the Configuration Registers.
Description: If enabled, this setting will override the Software PLL Enabled setting as it is mapped directly into the Configuration Registers instead of the Oscillator module registers. In this case, the Software PLL Enabled checkbox is unchecked and disabled. Depending on the multiplication factor (typically 3x or 4x) and if the selected clock can be used with the PLL, the Current System Clock (FOSC) is recalculated.
Additional Information: All time-sensitive peripherals (i.e. EUSART, Timers, PWM) are notified by this change and, if needed, some timings are recalculated. A warning is thrown in the Notifications tab if you have selected the PLL Enabled checkbox and if the System Clock Select is set to INTOSC.
Availability: Not usable for non-PLL frequencies or if the System Clock Select is set to INTOSC.

Software PLL Enabled

Summary: Enables the PLL via the Oscillator Registers.
Description: If the PLL Enabled setting is also enabled, this setting will be overridden and therefore unusable. In this case, the Software PLL Enabled checkbox is unchecked and disabled. Depending on the multiplication factor (typically 3x or 4x) and if the selected clock can be used with the PLL, the Current System Clock (FOSC) is recalculated.
Additional Information: All time-sensitive peripherals (i.e. EUSART, Timers, PWM) are notified by this change and, if needed, some timings are recalculated. A warning is thrown in the Notifications tab if you have selected the PLL Enabled checkbox and if the System Clock Select is set to INTOSC.
Availability: Not usable for non-PLL frequencies or if the System Clock Select is set to INTOSC.
