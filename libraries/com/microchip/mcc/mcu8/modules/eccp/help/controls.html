Enhanced Capture/Compare/PWM

ECCP Mode

Summary: ECCP Mode Selection drop-down,
Description: Selects between Capture, Compare and Enhanced PWM modes of operation of the ECCP module. Depending on the selection, only the corresponding settings below this drop-down are shown.
Additional Information: Some devices do not operate in any mode by default(ECCP Mode: disabled).
Availability: Always available.

Timer Selection

Summary: Assigns a timer to the ECCP module for mode-specific functions.
Description: Irrespective of the selected mode, the ECCP module will need a timer to perform the desired functions. In Capture mode, the timer is used in counter mode; in capture mode, the timer is left free-running to match a predefined value; in PWM mode, the timer period forms the base of the PWM period. When you make a timer selection a notification will pop-up in the notifications tab, letting you know you need to load and configure the timer you assigned to the ECCP module. Once you have loaded the timer module, the notification will change its type from Warning to Info, serving as a reminder you have assigned that timer module to the ECCP module. As a particular case, in the PWM mode, for each timer selected and loaded, the PWM Parameters section is updated with data (period, frequency, resolution) calculated based on information collected from the timer you selected.
Additional Information: For some devices, the timer selection is disabled due to a single selection being available for that ECCP module.
Availability: Conditioned by the ECCP mode selection. Not available when ECCP Mode is disabled.

Duty Cycle

Summary: Defines the PWM duty cycle as percentage.
Description: Based on the PWM period and resolution, MCC converts the percentage input into a ECCP period register value. The resolution of the input is up to one decimal place (i.e. xx.x format).
Additional Information: Using an out of range percentage input or non-numeric characters in the Duty Cycle input box will result in an invalid input which will render the box red until the input becomes valid again.
Availability: Conditioned by the ECCP mode selection. Not available when ECCP Mode is disabled, Capture or Compare.

Enhanced PWM Delay Counts

Summary: Sets the Programmable Dead-Band Delay
Description: Most useful when Enhanced PWM mode is set to Half Bridge. It prevents a current shoot-through on the bridge using a customizable delay between turning on the high-side and low-side switches. You can input the delay as a number of instruction cycles (up to 64 cycles). An instruction cycle is defined as Tcy = Fosc/4, where Fosc = System Clock. The PWM Delay label will be updated with the corresponding calculated time slice.
Additional Information: Using an out of range input will throw a warning in the Notifications tab. Using non-integer or non-numeric characters will result in an invalid input which will render the box red until the input becomes valid again.
Availability: On selected devices and only when ECCP Mode = Enhanced PWM.

Enhanced PWM Mode

Summary: Selects the PWM control mode.
Description: Offers different PWM control schemes, like independent output or n-bridge topologies. Depending on the Enhanced PWM mode and the PWM Pins Polarity selections, the appropriate pins are selected automatically in the Pin Manager window. Selecting the Single Output option allows you to use the Steering functionality which is disabled for all the other modes.
Availability: Only when ECCP Mode = Enhanced PWM. Options may vary.

Enhanced PWM Steering

Summary: Enable and configure the Pulse Steering functionality per each EPWM pin.
Description: If you only need to drive the same PWM output to one or multiple pins, you should use Pulse Steering. Once you enable steering via the Enable Steering checkbox, all steering-related settings are also enabled. You can now select the EPWM pins you want to drive the same PWM signal to, by checking the corresponding checkbox. Each checkbox is connected to the pin inside the Pin Manager window, so any selections in the Steering section will automatically select the pin in the Pin Manager window.
Availability: When ECCP Mode = Enhanced PWM and Enhanced PWM Mode = Single Output. On some devices the PWM Mode may have two Single Output options, with and without Steering.

Auto-shutdown

Summary: Enable and configure the Auto-shutdown feature.
Description: This is a safety-related feature that allows defining the system behavior in case of an electrical fault. Using the auto-shutdown drop-down you can select between various potential sources of failure and combinations of them, appealing to different scenarios. Once you've selected an auto-shutdown source (i.e. Auto-shutdown on != disabled) all the Auto-shutdown controls are made available for configuration. You can define the shut-down pin state and choose if the shutdown is followed by a restart automatically.
Availability: On selected devices and only when ECCP Mode = Enhanced PWM. Auto-shutdown sources may vary from one device to another.
