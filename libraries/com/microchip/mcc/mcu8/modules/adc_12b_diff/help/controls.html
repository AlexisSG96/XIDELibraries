Analog-to-Digital Converter (ADC)

ADC Clock Source

Summary: The ADC Clock Source dropdown box allows the user to configure the ADC sampling frequency and conversion timing.
Description: When altering the ADC Clock Source, the time to complete a one bit conversion (Tad) changes, therefore both the Sampling Frequency and Conversion Time are affected. MCC will recalculate all three values each time there's a change in the Clock Source. In the event that, after modifying the ADC Clock Source, the Tad value becomes smaller than a predefined minimum Tad value (typically 1us), a notification will be added to the MCC Notifications tab, warning that you should lower the sampling frequency by selecting an ADC Clock Source with a smaller frequency. The Clock Source options are primarily based on the System Clock (FOSC) and the internal RC oscillator.
Additional Information: For some devices, the ADC Clock interface will include an additional drop-down, Acquisition Time. This setting allows the user to select how many Tad's would the ADC require for data acquisition prior to the conversion process. It also has impact on the sampling frequency.
Availability: Always available.

Selected Channels Table

Summary: Displays the fixed internal channels (i.e. Temperature Sensor, FVR, DAC) and the user-selected channels.
Description: When selecting an ADC channel pin (ANx) from the Pin Manager window, that pin is added to both the Selected Channels table in the ADC User Interface and the Pins table in the Pin Module User Interface. In the latter, for each ADC channel pin, you can define a custom pin name which will also be shown in the ADC Selected Channels table. This custom name will serve as a reference in the generated ADC driver to help you configure and trigger a conversion for a specific channel.
Additional Information: Editing the ADC channel custom name is only available for external channels and only from the Pin Module User Interface.
Availability: Always available.
