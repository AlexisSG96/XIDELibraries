Comparator

Positive Input

Summary: Selects the signal to be connected to the positive terminal of the Comparator module.
Description: Depending on the device, this drop-down offers a range of options, each referring to a dedicated positive input pin (i.e. CIN0+), an internal signal from another peripheral (i.e. FVR, DAC) or the negative voltage reference of the microcontroller (Vss). Selecting a positive input CMP pin (i.e. CIN0+), will automatically be reflected in the Pin Manager window. The selection control is bidirectional, so if you select a positive input pin from the Pin Manager window, the corresponding option in the Positive Input dropdown will also be selected. A message box will pop-up if trying to deselect the positive input pin from the Pin Manager window. This is done in order to match the Positive Input register value, which does not allow a 'disabled' state, meaning that you will always have a signal connected to the positive input of the Comparator. Both the Positive Input signal and the Negative Input signal must be configured according to the application needs.
Availability: Always available. Signal options may vary.

Negative Input

Summary: Selects the signal to be connected to the negative terminal of the Comparator module.
Description: Depending on the device, this drop-down offers a range of options, each referring to a negative input pin (i.e. CIN0-) or the negative voltage reference of the microcontroller (Vss). Unlike the positive voltage reference input, the negative input pins are not dedicated to a Comparator module in particular - they are shared between all Comparator modules on the device. Selecting a negative input CMP pin (i.e. CIN0-), will automatically be reflected in the Pin Manager window. The selection control is bidirectional, so if you select a negative input pin from the Pin Manager window, the corresponding option in the Negative Input drop-down will also be selected. A message box will pop-up if trying to deselect the negative input pin from the Pin Manager window. This is done in order to match the Negative Input register value, which does not allow a 'disabled' state, meaning that you will always have a signal connected to the negative input of the Comparator. Both the Positive Input signal and the Negative Input signal must be configured according to the application needs.
Availability: Always available. Signal options may vary.

Edge-triggered interrupts

Summary: Enables the Comparator interrupt for rising and/or falling edge on the output signal
Description: By default, the CMP Interrupt is disabled (Enable Comparator Interrupt checkbox is unchecked), thus the two edge-triggers (Rising Edge and Falling Edge checkboxes) are disabled. Once you enable the CMP interrupt, you are also allowed to select whether the interrupt should be raised when the CMP output signal state transitions from low to high, high to low or both.
Availability: Depends on the device.
