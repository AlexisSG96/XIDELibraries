SRLATCH

Prescaler

Summary: The Prescaler drop-down box is used to reduce a higher frequency of the clock to a lower frequency
Description: The purpose of the prescaler is to allow the timer to be clocked at a desired value, hence modifying the Internal/External Clock value in the System Module and the Prescler in the SRLATCH module will result in a certain value displayed under the Clock tag. Thus, obtaining the desired Clock frequency requires both steps mantioned previously.
Availability: The Prescaler drop-down box is always available

Enable CMP Set/Reset

Summary: The Latch can be set or reset by the Comparator1 (CMP1) , Comparator2 (CMP2), SRI pin (check Enable Peripheral Set/Reset checkbox), software control (check Pulse Set/Reset Input checkbox ) or by the programmable clock (check Enable Set/Reset Clock checkbox ).
Description: Setting one of the CMP1 or CMP2 set/reset checkbox will trigger a hint in the notifications tab which will inform the user to load the CMP module used.
Availability: The checkboxes are always available.
