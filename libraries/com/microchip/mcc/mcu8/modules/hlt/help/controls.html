HLT

Timer Clock

Summary: These settings allow the user to select the clock source used by the HLT module.
Description: The timer clock settings set the clock parameters of the timer. The clock can be internal or external. Additionally, you can use a clock prescaler. Incrementation can happen on either clock edges.
Additional Information: If enabled, the prescaler can take values from 1:2 to 1:256. The clock source can be either internal or external. The external frequency can be manually entered.
Availability: The timer clock settings are always available.

Timer Period

Summary: Sets the timer period of the timer.
Description: The timer period setting box contain two main settings. A requested period and an actual period. Depending on the current selected setting, a minimum and a maximum period is displayed. Also, the actual period is the closest period the timer can use. The requested period should be between the maximum and the minimum value. If the reqested period is not within range, either the maximum or the minimum value will be selected automatically.
Additional Information: The actual period will be automatically calculated as integer multiples of the timer step. If finer steps are requried, the timer clock should be raised (and vice-versa). Please note that this will also impact the minimum and the maximum period.
Availability: The timer period setting is always available.
