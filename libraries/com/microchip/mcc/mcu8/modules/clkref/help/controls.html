Reference Clock

Duty Cycle

Summary: Selects the duty cycle amount for the output signal of the CLKREF module.
Description: This drop-down allows you to select between different predefined duty cycle percentages of the Reference Clock output signal. The selectable options can be seen as fractions of the CLKREF input signal (typically the System Clock).
Availability: Always available.

Clock Divider

Summary: Input Signal postscaler.
Description: This drop-down allows you to scale down the base clock signal using predefined scaling factors.
Additional Information: There are a couple of corner cases, where the Clock Divider might not work exactly as you intend it to. If you do not specify a scaling factor (Clock Divider = Baseclock), the duty cycle of the CLKREF output signal will match the one of the input signal, unless the Duty Cycle drop-down is set to 0%. Another special situation is the of Clock Divider = Baseclock/2 - in this case, if the Duty cycle is set to either 25% or 75% the accuracy of the actual duty cycle will depend on the accuracy of the clock source duty cycle.
Availability: Always available.
