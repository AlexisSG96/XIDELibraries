LCD

LCD Design Panel

The details of each of controls is given below:

  • Add Symbols from library: Launches a dialog where user can select one or multiple symbols to be added into design panel. When user clicks on symbol, a tick mark appears an indication that the symbol is selected. Once selection is completed press 'done' button.


  • Add External Image: Launches a file selection dialog where user can select single image to be added into design panel. The selected image automatically gets converted to single pixel symbol. PNG image with transparent background can be converted properly to similar looking symbol. For other format it'll be a dark color pixel.

  • Drag and Resize: Any symbol in the Design panel can be dragged to change its position or resized to scale it. The scale behavior can be invoked by positioning the cursor to bottom-left corner. The resize behavior can be invoked by clicking directly over the symbol.

  • Lock: 'Lock' button inhibits the drag and resize behavior. User is only allowed to click on the pixel and change MACRO and SEG, COM values.

  • Duplicate: This feature is mainly for saving effort in re-designing symbols. Select a symbol and duplicate. The duplicate symbol will look similar. User has to reconfigure SEG/COM and MACRO mapping for newly created symbol.

  • Delete: Select a symbol and press delete. All the settings related to the particular symbol namely MACRO, SEG, COM will be removed.

  • Reset Design: Invoking this button removes all symbols and respective SEG/COM mapping from the design panel. Before importing new design it's good to reset once. Reset only affects LCD deign panel it doesn't affect other LCD settings like (General Settings...)

  • Background Color: User can change the background color of the LCD panel. As the background color is of user convenience the color is not exported as part of LCD Design

  • Import LCD Design: The LCD design file (.lcd) can be imported by using this option.
    Below are the salient points of design file
    • The file exported by one device can be imported by another device i.e. the same design file works across devices.
    • If the imported design SEG, COM values mismatch the capabilities of the current device appropriate notification will be shown under notification tab. User can reassign to valid SEG, COM values.


  • Export LCD Design: LCD design can be exported in multiple ways. Below section covers them.
    • Design: This option launches a file dialog. User can select his design file name. The exported file design file only stores LCD panel design related attributes (Symbol, position, scale, MACRO, SEG, and COM). It doesn't store other module specific LCD settings.
    • Screenshot: This option exports the snapshot of the LCD design panel as (.png) image.


Configure Macro

Summary: How to configure MACRO
Description: Every symbol is associated with a macro. All symbols should have unique macro, as this is the #define constant that is generated in the output code. The GUI editor maintains uniqueness by appending _SYMXX. Only alphabets are allowed as macro.
Additional Information:


Availability: Only when a symbol or pixel is selected

SEG/COM Table Mapping

Summary: The SEG/COM table is a read only table which shows visual summary for the configured Pixels.
Description: It shows grouping of pixels with the symbol. If a pixel is clicked in the design panel corresponding Pixel is highlighted in the SEG/COM table. If a pixel group is clicked (Seven Segment Symbol) all the pixels corresponding to the symbol is highlighted.
Additional Information:

Availability: Always available

Backward Compatibility

Summary: The new LCD GUI maintains a backward compatibility with the old LCD projects.
Description: When the project is opened evenly distributed pixels and seven segments are drawn. The old pixel name (read only) is displayed under 'Old Macro' control. User can rename the new macro. To maintain backward compatibility with the old code, old macros are also generated as part of define.
Additional Information:
Availability: Only available for projects which saved using older MCC version.

General Settings

Summary: These settings allow the user to set the general settings of the LCD module.
Description: Select the multiplexing mode and the bias mode. According to the combination of the settings, the available commons and available pixels will be automatically calculated and displayed.
Additional Information:
Availability: The General Settings are always available.

Bias Settings

Summary: The pixels settings allow the user to configure the bias settings.
Description: Select the appropriate bias settings. A set of bias pins are selectable. Also, an internal bias reference can be enabled.
Additional Information:
Availability: The Bias Setting are always available.

Power Settings

Summary: The pixels settings allow the user to configure the power settings.
Description: There are two reference ladders available. These are noted with A and B. A set of power settings are individually selectable for both of them. A reference ladder time is also available.
Additional Information:
Availability: The Power Settings are always available.
