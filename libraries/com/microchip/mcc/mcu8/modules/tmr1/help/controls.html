Timer 1

Timer Clock

Summary: These settings allow the user to select the clock source used by the TMR1 module.
Description: The timer clock settings set the clock parameters of the timer. The clock can be internal or external. Additionally, you can use a clock pre-scaler.
Additional Information: If enabled, the prescaler can take values from 1:2 to 1:8. The clock source can be either internal or external. The external frequency can be manually entered, if this option is selected. You also have the possibility to enable synchronization with the internal clock.
Availability: The timer clock settings are always available.

Timer Period

Summary: Sets the timer period of the timer.
Description: Depending on the current selected setting, a minimum and a maximum period is displayed. Your period choice should be between the minimum and maximum values. If your choice is not withing range, you need the adjust your timer clock settings. Upon entering a period value, the period count will change accordingly. The period count is the period register. The calculated period shows the actual period that is the best achievable using the current clock settings.
Additional Information: The calculated period will be automatically calculated as integer multiples of the timer step. If finer steps are required, the timer clock should be raised (and vice-versa). Please note that this will also impact the minimum and the maximum period.
Availability: The timer period setting is always available.

Enable Gate

Summary: Sets the timer period of the timer.
Description: Timer1 can be configured to count freely or the count can be enabled and disabled using the Timer1 gate circuitry. This is also referred to as Timer1 gate count enable. Timer1 gate can also be driven by multiple selectable sources.
Additional Information: When Timer1 Gate Toggle mode is enabled, it is possible to measure the full cycle length of a Timer1 gate signal, as opposed to the duration of a single level pulse.When Timer1 Gate Single Pulse mode is enabled, it is possible to capture a single pulse gate event. The gate polarity selects the polarity of the gate enable. The gate signal source select box enables selection between the T1G signal and timer0 overflow.
Availability: The enable gate setting is always available.
