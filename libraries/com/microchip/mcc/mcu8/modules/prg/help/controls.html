PRG

Generator Mode

Summary: Sets the various generator modes.
Description: The PRG moduke can be operated in three ramp generator modes: Falling Voltage (slope compensation, Rising Voltage, Alternating Rising and Falling Voltage). You can select the voltage input source. Options may vary depending on the current selected device. Typical sources include DAC, FVR, and OPA output. The slope rate is also adjustable.
Additional Information:
Availability: The Generator Mode Settings are always available.

Rising Edge

Summary: Sets the timing source for the rising edge.
Description: You can select the timing source from various options. Options may vary depending on the current selected device. Typical sources include CCP, PWM, and Comparator output. The sensitivity can be either level sensitive or edge sensitive. Polarity can be active high or active low.
Additional Information:
Availability: The Rising Edge Setting are always available.

Falling Edge

Summary: Sets the timing source for the falling edge.
Description: Select the appropriate bias settings. A set of bias pins are selectable. Also, an internal bias reference can be enabled.
Additional Information:
Availability: The Falling Edge Setting are always available.
