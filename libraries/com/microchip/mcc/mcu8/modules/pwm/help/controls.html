Pulse Width Modulation

Timer Selection

Summary: Assigns a timer to the PWM module.
Description: The PWM module will need a timer to perform the desired functions. In Capture mode, the timer is used in counter mode; in capture mode, the timer is left free-running to match a predefined value; in PWM mode, the timer period forms the base of the PWM period. When you make a timer selection a notification will pop-up in the notifications tab, letting you know you need to load and configure the timer you assigned to the PWM module. Once you have loaded the timer module, the notification will change its type from Warning to Info, serving as a reminder you have assigned that timer module to the PWM module. For each timer selected and loaded, the PWM Parameters section is updated with data (period, frequency, resolution) calculated based on information collected from the timer you selected.
Additional Information: For some devices, the timer selection is disabled due to a single selection being available for that PWM module.
Availability: Always available.

Duty Cycle

Summary: Defines the PWM duty cycle as percentage.
Description: Based on the PWM period and resolution, MCC converts the percentage input into a PWM period register value. The resolution of the input is up to one decimal place (i.e. xx.x format).
Additional Information: Using an out of range percentage input or non-numeric characters in the Duty Cycle input box will result in an invalid input which will render the box red until the input becomes valid again.
Availability: Always available.
