Real Time Clock/Calendar (RTCC)

RTCC Clock Source

Summary: The RTCC clock source can be changed from the System Module, Register View tab in the CONFIG3L register, if available. It cannot be edited in the RTCC module viewer.
Description: The RTCC module is intended to be clocked by an external Real-Time Clock (RTC) crystal oscillating at 32.768 kHz, but an internal oscillator can be used.
Availability: The RTCC clock source is always available.

RTCC Output

Summary: The RTCC Output allows the selction of the Alarm Pulse, Seconds Clock or Source Clock on the RTCC pin.
Description: The RTCC Output drop down box can be edited when selecting the Enable RTCC Output checkbox.
Availability: Always available.

Alarm Enable

Summary: The setting of the alarm is done by entering the desired information in the corresponding boxes in the Alarm panel.
Description: By enabling the Alarm Enable checkbox the information about the alarm time, frequency, count and repeat can be modified. Clicking the clock symbol on the right of the Alarm Time pops up a calendar which allows the selection of a year, a month, a day, or by sliding the controls on the bottom, even the hour and the minute. The selection is recorded only after clicking on the check mark on the top right of the pop-up.
Additional Information: The Intial Time it is set following the same steps as for the Alarm Time setting.
Availability: Always available.
