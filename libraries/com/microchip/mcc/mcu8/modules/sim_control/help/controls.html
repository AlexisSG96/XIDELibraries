Simultaneous PWM Control

Enable Simultaneously

Summary: Synchonizes PWM outputs at start
Description: The outputs of some PWM modules can be either enabled independently or simultaneously. The SIM_CONTROL graphical interface allows you to select simultaneous PWM start through the Enable Simultaneously checkbox. After doing this, the individual signal checkboxes will also be enabled and you will be able to select which PWM outputs you want to drive simultaneously. A warning will be thrown in the Notifications tab for each PWM signal you select, letting you know that you need to enable the corresponding PWM module and configure it. Once you have enabled a PWM module, the afferent warning will disappear.
Additional information: Keep in mind that you can drive simultaneously only some PWM modules. Your device may contain different types of PWM modules (i.e. CCPs, ECCPs) which do not share this functionality.
Availability: Always available. Serves as condition to enable PWM signals independently.

Reload Simultaneously

Synchonizes PWM outputs at reload
Description: For some PWM modules, the Duty Cycle, Period, Phase and Offset registers are double-buffered. Writing to these registers is, therefore, not immediate but delayed until arming conditions are valid. In the graphical interface of the PWM module you can select between immediate or triggered arming in the PWM Reload section. The SIM_CONTROL graphical interface allows you to select simultaneous PWM reload through the Reload Simultaneously checkbox. After doing this, the individual signal checkboxes will also be enabled and you will be able to select which PWM outputs you want to reload simultaneously. A warning will be thrown in the Notifications tab for each PWM signal you select, letting you know that you need to enable the corresponding PWM module and configure it. Once you have enabled a PWM module, the afferent warning will disappear.
Additional information: Keep in mind that you can reload simultaneously only some PWM modules. Your device may contain different types of PWM modules (i.e. CCPs, ECCPs) which do not share this functionality.
Availability: Always available. Serves as condition to reload PWM outputs independently.
