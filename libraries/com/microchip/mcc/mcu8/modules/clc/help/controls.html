CLC Module

Hardware Settings

Summary: Sets the various hardware settings of the CLC module
Description: The hardware settings allow the user to enable or disable a set of hardware related settings of the CLC module. Options include an general CLC enable/disable, rising and falling interrupts, and a geneal CLC output.
Additional Information:
Availability: The Hardware Settings is always available.

Mode

Summary: Sets the specific logic mode of operation.
Description: The mode setting allow the user to choose the exact functionality that is required. The selected logic type is graphically displayed. At any time a set of four user-assignable input are available. The user can route them to various signal sources. The exact signal source selectable may depend on the current selected microcontroller. The Configurable Logic Cell (CLC) provides programmable logic that operates outside the speed limitations of software execution. The logic cell takes up to 16 input signals and through the use of configurable gates reduces the 16 inputs to four logic lines that drive one of eight selectable single-output logic functions. The CLC peripheral has four sections that need to be setup before it can be used. This involves setting up 8 registers in your software program. Once these registers are setup, the CLC will run independent of software control until the registers are changed via software.
