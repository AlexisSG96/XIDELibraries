Enhanced Universal Control Area Network (ECAN)

CAN Bus Settings

Summary: These settings control the CAN bus speed and transmission settings.
Description: You have the options to select the CAN BUS speed. Depending on the processor frequency and the CAN BUS speed a series of Tq values are being calculated and recommended. The software will automatically select the timing parameters for your current settings. The
Additional Information: Based on the current Time Quanta selected, the Propagation Segment, the Phase Segment 1, and Phase Segment 2 are being calculated so the Sample Point is as close to 80% as possible. The user can manually override these settings, provided that the sum of the segments matches the selected Time Quanta.
Availability: The CAN BUS settings are always available.

General Settings

Summary: These settings allow you to select the wake-up options.
Description: The CAN Activity wake-up setting allows you to enable the wake-up on any CAN activity. The CAN line filter wake-up enable allows you to enable the wake-up on CAN Line Filter.
Additional Information: -
Availability: The General Settings are always available.

CAN Mode and CAN IDs

Summary: Selects the CAN mode of operation.
Description: Allows you to select the CAN mode of operation. Select between Mode0, Mode1, and Mode2. Depending on the current selected mode, the CAN hardware module will be configured different. Add the CAN Id,Mask, Filter, and Buffer information and then clock on the Add button. If a row needs to be deleted, select that row and press the Remove button.
Additional Information:
Availability: The Mode Settings are always available.
