CWG

Hardware Settings

Summary: Sets the various CWG hardware operating settings.
Description: The input source of the CWG module can be selected from various options. Options may vary depending on the current selected device. Typical options include comparator output, CCP. The output mode selects different modes of operation. The clock source can also be selected.
Additional Information:
Availability: The Hardware Settings are always available.

Events

Summary: Sets the CWG events parameters.
Description: This allows the user to select the number or rising or falling counts. Upon selection, the minimum and maximum dead-band time is being calculated.
Additional Information:
Availability: The Events Settings are always available.

Output Pins

Summary: These settings enable the CWGx output pins.
Description: Steering data (if the option is activated) can be selected to high or low. Polarity can also be selected between inverted and non-inverted. These options are individual for each pin.
Additional Information:
Availability: The Output Pins Settings are always available.

Auto Shut Down

Summary: Sets the the automatic shutdown settings.
Description: A series of auto-shutdown sources can be selected. Typical inputs are comparator outputs. An auto-restart option is available. Shutdown states can be selected for CWGA/CWGC and CWGB/CWGD set of pins.
Additional Information:
Availability: The Auto Shut Down options are always available.
