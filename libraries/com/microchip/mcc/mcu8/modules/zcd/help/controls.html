Zero Cross Detection (ZCD)

ZCD Logic Output Polarity

Summary: The Logic Output Polarity dropdown box can be set to invert or not the ZCDxOUT bit.
Description: The output of the ZCD module is monitored by the state of the ZCDxOUT bit which can be controlled by setting the ZCDxPOL bit. Changing the ZCDxPOL bit will cause an interrupt, regardless of the level if the ZCDxEN bit.
Availability: The Logic Output Polarity is always available.
