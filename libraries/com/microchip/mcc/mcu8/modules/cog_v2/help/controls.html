COG

Hardware Settings

Summary: Sets the various COG hardware operating settings.
Description: Allows the user to select the general hardware settings, such as the clock source and an interrupt enable.
Additional Information:
Availability: The Hardware Settings are always available.

Events

Summary: Sets the COG events parameters.
Description: Select the parameters for the rising event and falling event. The two events are independent and there are separate settings for both of them. For both the rising and falling event you can select the source. Options may vary depending on the current selected device. The event source may be edge sensitive or level sensitive. Typical options include Comparator Output, CCP, and COG. Other options include a dead-band count, blanking count and a phase delay.
Additional Information:
Availability: The Events Settings are always available.

Output Pins

Summary: These settings enable the COGx output pins.
Description: There are separate options for both outputs of the COG module. Both outputs can be enabled or disabled and the polarity is user-selectable.
Additional Information:
Availability: The Output Pins Settings are always available.

Auto Shut Down

Summary: Sets the the automatic shutdown settings.
Description: The auto-shutdown section controls different auto-shutdown sources and options. Typical sources include comparators. The user also has the option to select the override logic level for both outputs of the COG.
Additional Information:
Availability: The Auto Shut Down options are always available.
