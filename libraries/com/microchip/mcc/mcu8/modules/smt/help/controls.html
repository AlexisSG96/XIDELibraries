SMT

Operating Mode

Summary: Sets the various operating modes.
Description: The SMT has different operating modes. The modes of operations include: Timer,Gated Timer,Period and Duty Cycle Acquisition, High and Low Time Measurement,Windowed Measurement, Gated Windowed Measurement,Time of Flight, Capture, Counter, Gated Counter, Windowed Counter.
Additional Information: The Aquisition mode can be either a Single Acqusition mode or Repeat Data Aquisition.
Availability: The Operating Mode Settings are always available.

Clock

Summary: Sets the clock source.
Description: Various clock sources are selectable along with a prescaler and input polarity.
Additional Information:
Availability: The Clock Setting are always available.

Signal

Summary: Sets the various signal sources.
Description: Various input sources are selectable. Options may vary depending on the current selected device. Typical sources include Comparator, CCP, CLC, PWM, Timer Postscaled, Timer overflow, and ZCD. The polarity can either be on a rising edge or falling edge.
Additional Information:
Availability: The Signal source is available only in some operating modes.

Window

Summary: Sets the various window settings.
Description: The window source is available only in some operating modes. Various input sources are selectable. Options may vary depending on the current selected device. Typical sources include AT, Comparator, CLC, PWM, Timer Postscaled, Timer overflow, and ZCD. The polarity can either be on a rising edge or falling edge.
Additional Information:
Availability: The Window options are available only in some operating modes.
