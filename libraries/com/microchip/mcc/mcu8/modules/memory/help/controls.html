Memory DataEE Routines

Summary: Data EEPROM is a non-volatile memory, which stores the information even after the power is turn off.
Description: By default Flash Routines are always included because some enhanced mid-range PIC devices lack the data EEPROM found on other devices, thus the Flash Memory is available as a non-volatile memory.
Availability: The Add DataEE Routines checkbox is available only for device which have EEPROM. PIC16F1XXX device family lacks EEPROM
