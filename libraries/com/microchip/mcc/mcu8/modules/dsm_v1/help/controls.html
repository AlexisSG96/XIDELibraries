DSM

Modulation Output

Summary: These settings allow the user to select the modulation output of the DSM module..
Description: Enables of disables the output pin, polarity and slew rate control.
Additional Information:
Availability: The Modulation Output settings are always available.

Modulation Source

Summary: Allows you to select the source of the modulation.
Description: various options are available depending on the current selected device. Options typically include Manual, CCP, USART, and MSSP.
Additional Information:
Availability: The Modulation Source setting are always available.

High Carrier

Summary: The software settings configure the software settings of the peripheral driver.
Description: Options typically include: CCP, MDCIN, Vss, and Ref Clock. There are options to enable the carrier pin, invert polarity and synchronize with carrier.
Additional Information:
Availability: The High Carrier settings are always available.
