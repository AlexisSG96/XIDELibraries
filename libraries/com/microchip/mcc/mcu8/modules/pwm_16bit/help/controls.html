Pulse Width Modulation

PWM Mode

Summary: Selectable PWM functions.
Description: Depending on the device, the PWM module can be configured to provide a Standard PWM, Set/Toggle Output on phase count match or Center-Aligned Output. Selecting/Deselecting the Center-Aligned mode enables/disables the Match On and Trigger Source drop-downs in the PWM Offset section.
Availability: Always available.

Clock Source

Summary: The input clock signal to the PWM module.
Description: The Clock Source drop-down allows you to select the base clock signal for the PWM module. Options may vary among different devices and may include derivations of Fosc, Timers, or external clock signals. Any Clock Source change results in MCC attempting to recalculate its parameters to match the requested Period value. Both the Period and Offset ranges are recalculated with each Clock Source selection. A warning will be added to the Notifications tab if the requested Period cannot be reached with the new Clock Source selection.
Additional Information: You can further refine a Clock Source using the Prescaler drop-down.
Availability: Always available.

Clock Prescaler

Summary: A prescaler for the input clock signal to the PWM module.
Description: You can further refine a Clock Source using the Prescaler drop-down. Both the Period and Offset ranges are recalculated with each Clock Prescaler selection.
Availability: Always available.

Offset Match

Summary: The PWM Timer slope where the offset match occurs
Description: When The PWM module is configured to operate in the Center-Aligned mode, you are able to select whether the offset match will occur on the rising or falling counter slopes of the PWM Timer.
Additional Information: Any PWM Mode selection other than Center-Aligned will disable the Offset Match On drop-down.
Availability: Always available.

Offset Mode

Summary: Defines a timing relationship to another PWM signal on the device.
Description: Using this drop-down, you are able to synchronize the output of the current PWM module to another PWM module output on the device, introducing different offsets. If you don't need to synchronize to another internal PWM signal, you can operate in the Independent Run mode of the PWM module.
Additional Information: The Trigger Source drop-down is disabled when the PWM Offset Mode is set to Independent Run.
Availability: Always available.

Offset Trigger Source

Summary: Selects the offset signal to match against.
Description: This drop-down offers several internal PWM signals you can use as offset signals to the current PWM module.
Availability: Disabled when the PWM Offset Mode is set to Independent Run.

Period

Summary: The PWM Period.
Description: Use this input box to specify the desired PWM period as a time slice. MCC will do all necessary calculations attempting to reach the value you set, given the selected Clock Source The Period Count is also recalculated for each change in Period or Period range. Make sure the Period input uses the format [Number][Units], where Units = {ns, us, ms, s} and falls within the range specified.
Additional Information: Using an input which is out of the specified range will throw a waning in the Notifications tab. Using other characters than the format above will be considered as invalid input and the input box will blink red until a valid input is added or the focus is lost by the input box.
Availability: Always available.

Phase

Summary: The Phase of the PWM signal, expressed in degrees.
Description: The predefined Phase Count value is constantly compared to the PWMTMR timer counter. You can define the Phase Count by expressing it in degrees (which is more suitable from an application viewpoint) and MCC will convert it to Phase Count. Make sure the Phase input is an unsigned integer value and falls within the range specified (0 - 360 degrees).
Additional Information: Using an input which is out of the specified range will throw a waning in the Notifications tab. Using non-integer values or non-numeric characters will be considered as invalid input and the input box will blink red until a valid input is added or the focus is lost by the input box.
Availability: Always available.

Offset

Summary: The offset between the PWM signal of the current module and a signal from another PWM module
Description: Use this input box to specify the desired Offset time slice. MCC will do all necessary calculations attempting to reach the value you set, given the selected Clock Source Make sure the Offset input uses the format [Number][Units], where Units = {ns, us, ms, s} and falls within the range specified.
Additional Information: Using an input which is out of the specified range will throw a waning in the Notifications tab. Using other characters than the format above will be considered as invalid input and the input box will blink red until a valid input is added or the focus is lost by the input box.
Availability: Always available.

Duty Cycle

Summary: Defines the PWM duty cycle as percentage.
Description: Based on the PWM period and resolution, MCC converts the percentage input into a PWM period register value. The resolution of the input is up to one decimal place (i.e. xx.x format).
Additional Information: Using an out of range percentage input or non-numeric characters in the Duty Cycle input box will result in an invalid input which will render the box red until the input becomes valid again.
Availability: Always available.
