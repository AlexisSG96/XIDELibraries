External Interrupts (EXTINT)

Interrupt Priority

Summary: The external interrupt table shows the interrupt bits available, accessible from the pin map.
Description: By default the INT0 bit has highest priority assigned, on the devices that allow multiple external pins. The edge detect can be changed from the Register view to either Falling or Rising Edge, for each external interrupt pin.
Additional Information: On the PIC16 families there is only one external interrupt pin found under the name of INT.
Availability: The external interrupt table is always available.
