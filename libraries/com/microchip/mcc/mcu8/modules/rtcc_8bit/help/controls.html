Real Time Clock/Calendar (RTCC)

RTCC Clock Source

Summary: The RTCC module can be clocked by either an external RTCC crystal oscillator, MFINTOSC/16 or via ZCD.
Description: The module can be clocked by an external RTCC crystal oscillating at 32.768kHz, MFINTOSC/16(31.25kHz) or via the ZCD at 50Hz or 60Hz. Each clock selection has a fix prescaler in order to generate the required 1/2 clock needed by RTCC. The clock source can be accessed in Register View via RTCCLKSEL bit field from RTCCON register.
Availability: The RTCC clock source is always available.

Alarm Enable

Summary:

The setting of the alarm is done by entering the desired information in the corresponding boxes in the Alarm panel.

This alarm is configurable from half second to one year.
Description:

By enabling the Alarm Enable checkbox the information about the alarm time, frequency, count and repeat can be modified. Clicking the clock symbol on the right of the Alarm Time pops up a calendar which allows the selection of a month, a day, or by sliding the controls on the bottom, even the hour and the minute. The selection is recorded only after clicking on the check mark on the top right of the pop-up.

The Intial Time it is set following the same steps as for the Alarm Time setting.

Availability: Always available.
