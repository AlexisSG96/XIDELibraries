Slope Compensator (SCOMP)

Timing Select

Summary: The Timing Select dropdown box allows the selections between four different outputs.
Description: The output connection of the SCOMP module can be connected to either Comparator 1, Comparator 2, COG 1 or COG 2. The selection of any output will generate a hint in the notifications tab to inform the user to also configure the selected module coresponding to the output.
Availability: The Timing Select is always available.

Slope Value

Summary: The Slope Value is the result of the calculated Slope Compensator Value, expressed in volts per microsecond is computed using the above equation.
Description: Selecting a certain slope value the Current Setting value is updated, based on the information in the Slope Compensator Current Setting table, available in the datasheet. When selecting from the Range dropdown-box high or low and the Slope value is not in range, a hint will appear in the Notifications tabs.
Availability: The Slope Value is always available.
