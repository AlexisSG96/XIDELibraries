Enhanced Universal Synchronous Asynchronous Receiver Transmitter (EUSART)

EUSART Baud Rate

Summary: The Baud Rate dropdown box allows the user to specify the desired EUSART baud rate. The use can select one of the predefined baud rates, or enter a numeric value for the desired baud.
Description: After a baud rate is selected, MCC will calculate all of the possible baud rates for the device based on the current system clock setting. From these possible baud rates, the baud rate which is closest to the desired baud rate will be selected. The registers that control the baud rate will be updated to produce the actual baud rate.
Additional Information: The baud rate can be set using two different methods. The baud rate can be selected on the Easy Setup tab, or on the Registers tab, the registers that control the baud rate can be set directly. The behavior of MCC is different depending on the method used to set the baud rate.Setting the baud rate from the Easy Setup window, MCC will always use the desired baud rate setting to findan actual baud that is closest to the desired baud rate. If the System Clock is later changed in MCC, the desired baud rate will not change.MCC will again search for an actual baud that is closest to the desired baud, based on the new System Clock value.Setting the baud rate by configuring the registers from the Registers window, MCC will discard desired baud rate. The baud rate dropdown box will display the actual baud rate that will be produced from the register settings. The register settings that control the baud rate will not be changed by MCC. If The System Clock is later changed in MCC,the new actual baud rate will be calculated and shown in the Baud Rate dropdown box.
Availability: The Baud Rate Dropdown is always available.

Software Transmit/Receive Buffer Size

Summary: The Software Transmit and Receive Buffer size dropdown boxes allow the buffer size to be specified
Description: When using interrupts to process the EUSART data, buffers must be provided to allow the interrupt service routine to store data to be sent or received. These controls specify the size of the ring buffer that will be used.
Additional Information:
Availability: The Software Transmit and Receive Buffer Size dropdown boxes are only available if the 'Enable EUSART Interrupts' check box is checked.
