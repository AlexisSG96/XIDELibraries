FatFS System Configuration

This section configures the system parameters of the library such as time stamping, RTOS support features, etc.

exFAT Enable/Disable

Description: Enables/Disables support for exFAT in the file system. exFAT support is required for SDXC cards, but most USB drivers and SD/SDHC cards are FAT/FAT32 formated and do not require exFAT. This option will increase resource usage, but will increase drive/card support and extend longevity of the solution.
This corresponds to the FF_FS_EXFAT option.

Timestamping Options

Summary: Select how the file system gets the timestamp to use when creating or modifying a file.
Description: There are two main options for the timestamping features, RTC and fixed timestamp. When the fixed timestamp option is selected, all files are stamped with the same date. Use the calendar date selector to pick the date you want used. When the RTC option is selected, the FatFs library will call the get_fattime() function to get the current time. It is the users responsibility to implement this function. When this option is selected, example files ff_time.c/.h are added to the project to help in implementing this functionality.
This option is disabled when in read-only mode.
This corresponds to the following options: FF_FS_NORTC, FF_NORTC_MON, FF_NORTC_MDAY, FF_NORTC_YEAR

Re-entrant

Description: Enables/Disables support for re-entrance in the file/folder operation functions. Select this option if your application need to be able to call these functions from different threads/contexts concurrently. If enabled, user is responsible for implementing the required functions/objects for the thread-safe operations.
This corresponds to the FF_FS_REENTRANT option.

Timeout

Description: Defines the number of system ticks to wait for an operation to complete when in re-entrant mode. Not used when re-entrant is disabled..
This corresponds to the FF_FS_TIMEOUT option.

Timeout

Description: Defines the number of system ticks to wait for an operation to complete when in re-entrant mode. Not used when re-entrant is disabled.
This corresponds to the FF_FS_TIMEOUT option.

Sync Object

Description: Defines what the OS sync object name is. This allows the FatFs library to interact with the file system. This option is only used when re-entrance is enabled.
This corresponds to the FF_SYNC_t option.
