FatFS Demo Configuration

Configures the demo/example code generation.

Generate Example Demo

Summary: Enables/disables the generation of example files that show how to write a file to a drive using FatFs.
Description: This option allows the user to select if they want the example demo files generated into the project or not. For some devices the user may still need to call the example initialize and tasks routines to the main.c file for them to run. On some devices this is automatically added.
