FatFS Long File Name Options

This section contains the configuration options for the long file name support. These options are not applicable if long file name support is disabled.

Long File Name Support Options

Description: Enables/disables long file name (LFN) support. Also determines how memory for long file name buffers are allocated.
This corresponds to the FF_USE_LFN option.

Maximum Long File Name Length

Summary: Defines the length of the long file name buffer (maximum file length supported).
Description: Defines the length of the long file name buffer (maximum file length supported). It is suggested to keep this set to 255 (the maximum allowed) in order to support all possible files. This has an impact on the RAM foot print of the library. This feature is only enabled when LFN is enabled.
This corresponds to the FF_MAX_LFN option.

Long File Name Unicode Encoding

Description: Defines the Unicode format used to encode strings in the library API. This feature is only enabled when LFN is enabled.
This corresponds to the FF_LFN_UNICODE option.

Long File Name Buffer Length

Description: Defines the length of the long file name buffer (maximum file length supported) for the currently selected Unicode formatting. This feature is only enabled when LFN is enabled.
This corresponds to the FF_LFN_BUF option.

Short/Alternate File Name Buffer Length

Description: Defines the length of the short/alternate file name buffer (maximum file length supported) for the currently selected Unicode formatting. This feature is only enabled when LFN is enabled.
This corresponds to the FF_SFN_BUF option.

String Functions Unicode Encoding

Description: Defines the Unicode encoding used in the string library functions (f_printf, f_puts, f_gets, fputc).
This corresponds to the FF_STRF_ENCODE option.
