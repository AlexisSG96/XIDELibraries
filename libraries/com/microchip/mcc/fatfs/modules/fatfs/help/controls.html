FatFS

FatFS Home Page

FatFS is a generic FAT file system developed outside of Microchip Technology Inc. For more information regarding FatFs including module settings documentation, FatFs application notes, and file system API documentation please visit the FatFs Home page:
http://elm-chan.org/fsw/ff/00index_e.html

Or click the FatFS Home Page link in the External Links reference of this help file.

This document will focus on this modules controls for configuring the FatFs library and not on the FatFs library itself.

Demo Configuration

Configures the demo/example code generation.

Function/Optimization Configuration

Allows the user to select which features of the FatFs library to include allowing the user to remove unused features to reduce code footprint. This section also allows user to configure how some of the functions operate.

Locale and Namespace Configuration

Configures the buffer sizes for the library, thus impacting the RAM footprint. The section also contains information about the formatting of the strings used to reference files/folders on the drive.

Drive/Volume Configuration

Configures what types of drives are supported, how they are supported, and creates a linkage between the FatFs library and the physical layer driver that implements the communication to that drive.

System Configuration

This section configures the system parameters of the library such as time stamping, RTOS support features, etc.
