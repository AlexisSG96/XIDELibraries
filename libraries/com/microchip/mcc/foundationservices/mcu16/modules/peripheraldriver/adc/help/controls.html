ADC TAD Calculation

Summary: Displays the calculated TAD value for the ADC
Description: When the ADC Clock Source is altered, the time taken for one bit to complete conversion (TAD) changes. TAD value also changes based on the Conversion Clock TCY value. MCC will recalculate the TAD each time there's a change in the Clock Source or the Conversion Clock. In the event that the Tad value becomes smaller than a predefined minimum Tad value (typically 1us), a notification will be added to the MCC Notifications tab. The Clock Source options are primarily based on the System Clock (FOSC) and the internal RC oscillator.
Availability: Always available

ADC Selected Channels table

Summary: Displays the fixed internal channels (i.e. Temperature Sensor, FVR, DAC) and the user-selected channels.
Description: When an ADC channel pin (ANx) is selected from the Pin Manager window, the pin is added to both the Selected Channels table in the ADC User Interface and the Pins table in the Pin Module User Interface. In the latter, a custom pin name can be defined for each ADC channel pin which will also be reflected in the ADC Selected Channels table. This custom name will serve as a reference in the generated ADC driver to help configure and trigger a conversion for a specific channel. Multiple ADC channels can be scanned by enabling the Scan Enable checkbox. If any channel has been enabled for scan, the CSCNA bit is set. When an ADC channel is enabled for scan, the corresponding AD1CSSH/L bit is set.
Additional Information: Editing the ADC channel custom name is only available for external channels and only from the Pin Module User Interface.
