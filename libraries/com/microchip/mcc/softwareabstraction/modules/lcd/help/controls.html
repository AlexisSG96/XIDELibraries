LCD

LCD Design Panel

The details of each of controls is given below:

  • Add Symbols from library: Launches a dialog where user can select one or multiple symbols to be added into design panel. When user clicks on symbol, a tick mark appears an indication that the symbol is selected. Once selection is completed press 'done' button.


  • Add External Image: Launches a file selection dialog where user can select external custom single image to be added into design panel. The selected image automatically gets converted to single pixel symbol. PNG image with transparent background is supported which can be converted to symbol. If the selected image format is not supported then it will be converted to a dark color pixel.

  • Drag and Resize: Any symbol in the Design panel can be dragged to change its position or resized to scale it. The scale behavior can be invoked by positioning the cursor to bottom-left corner. The resize behavior can be invoked by clicking directly over the symbol.

  • Lock: 'Lock' button inhibits re-positioning drag and resizing of the symbol. User is only allowed to click on the pixel and change NAME and SEG, COM values.

  • Duplicate: This feature is mainly for replicating the symbols. Select a symbol and duplicate. User has to reconfigure SEG/COM and NAME mapping for newly created symbol.

  • Delete: Select a symbol and press delete. All the settings related to the particular symbol NAME, SEG, COM will be removed.

  • Reset Design: Invoking this button removes all symbols and respective SEG/COM mapping from the design panel. Before importing new design it's good to reset once. Reset only affects LCD deign panel it doesn't affect other LCD settings like (Basic Configuration Settings...)

  • Background Color: User can change the background color of the LCD panel. As the background color is of user convenience the color is not exported as part of LCD Design

  • Import LCD Design: The LCD design file (.lcd) can be imported by using this option.
    Below are the salient points of design file
    • The file exported by one device can be imported by another device i.e. the same design file works across devices.
    • If the imported design SEG, COM values mismatch the capabilities of the current device appropriate notification will be shown under notification tab. User can reassign to valid SEG, COM values.


  • Export LCD Design: LCD design can be exported in multiple ways. Below section covers them.
    • Design: This option launches a file dialog. User can select his design file name. The exported file design file only stores LCD panel design related attributes (Symbol, position, scale, SEG, and COM). It doesn't store other module specific LCD settings.
    • Screenshot: This option exports the snapshot of the LCD design panel as (.png) image.


Configure Symbol

Summary: How to configure Symbol
Description: Every symbol is associated with a name. All symbols/pixels should have unique name, as this is the #define constant that is generated in the output code.
Additional Information:


Availability: Only when a symbol or pixel is selected

Configure Multipixel-ed Symbol

Summary: How to configure Multipixel-ed Symbol
Description: Symbols such as 7-Segment, Battery and Wheel have multiple pixels to be configured. User can change the name of the symbol as a whole and a suffix (Ex. _A, _B,...) for each pixel along with the user give name will be appended automatically to maintain uniqueness in naming. User cannot change/edit the name of individual pixel inside the symbol.
Additional Information:

Availability: Only when a symbol is selected

SEG/COM Table Mapping

Summary: The SEG/COM table is a read only table which shows visual summary for the configured Pixels.
Description: It shows grouping of pixels with the symbol. If a pixel is clicked in the design panel corresponding Pixel is highlighted in the SEG/COM table. If a pixel group is clicked (Seven Segment Symbol) all the pixels corresponding to the symbol is highlighted.
Additional Information:

Availability: Always available

Basic Configuration Settings

Summary: These settings allow the user to set the basic settings of the LCD module.
Description:

Select the multiplexing mode. According to the multiplex mode, the available bias modes are listed and maximum pixels will be automatically calculated.

Select clock source. According to clock source, the avilable frame frequencies are listed.

Additional Information:

Availability: The Basic Configuration Settings are always available.
Waveform Type: The default waveform type for LCD is set to Type - A implicitly.

Bias Control Settings

Summary: These settings allow the user to set Bias/Contrast control settings of the LCD module.
Description:

Bias Control consists of its Resistor ladder selection and Contrast control. In case the resistor ladder is Internal, available contrast settings are
listed from Maximum Possible contrast [minimum resistance] to Minimum Possible contrast [maximum resistance]

Additional Information:

Availability: Internal Contrast Control is available when Bias Control is set to Internal Resistor Ladder.
